# babili-webpack-plugin

Babili - A babel based minifier - https://github.com/babel/babili

## Install

```sh
npm install babili-webpack-plugin --save-dev
```

## Usage

```js
// webpack.config.js
const BabiliPlugin = require("babili-webpack-plugin");
{
  plugins: [
    new BabiliPlugin()
  ]
}
```
